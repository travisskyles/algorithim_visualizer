# Algorithim Visualizer

A React app to visually show a variety of pathfinding and maze creation algorithms.

View at [skyles-algo-visualizer.netlify.app](skyles-algo-visualizer.netlify.app)

## Current Algorithms

### Pathfinding

- Dijkstras
- A \*
- Breadth First

### Maze Creation

- Binary Tree
- Depth First
